## 职位详情

### 1. 高级全栈工程师
**岗位职责：**
1. 参与 AI 产品开发，负责前后端系统的架构设计，优化高并发、低延迟场景下的稳定性与扩展性。
2. 编写高质量、可维护的核心代码，通过 vibe coding、自动化 CI/CD 体系，提升团队工程效能。

**岗位要求：**
1. 计算机相关专业本科及以上学历，**5 年以上**全栈开发经验。
2. 能够独立完成项目的架构设计与技术选型工作。
3. 熟练掌握 **Golang、Python、Java** 等后端主流编程语言之一，深入理解微服务架构、分布式系统设计。
4. 熟练掌握 **React、Vue、Angular** 等主流前端架构之一。
5. 拥有良好的沟通能力和团队协作精神。

### 2. 后端开发工程师
**岗位职责：**
1. 设计、开发、部署和维护后端系统，确保系统的稳定性、可扩展性和安全性。
2. 进行代码评审，提高代码质量。

**岗位要求：**
1. 具有 **2 年以上**的后端开发经验，熟练掌握至少一种 Python、Golang 或 Java 等编程语言及常用的后端框架。
2. 能够独立完成项目的开发和调试工作。
3. 熟悉数据库设计和优化，能编写 SQL，熟悉 **Redis、ETCD** 等常用缓存技术，熟悉 **Docker 和 Kubernetes**。
4. 有多模态产品（如音视频，多媒体等）相关开发经验者优先。
5. 拥有良好的沟通能力和团队协作精神。

### 3. 前端开发工程师
**岗位职责：**
1. 负责 Web 应用的前端开发工作，包括但不限于页面版面、交互实现、动画效果等。
2. 根据产品需求和技术规范，编写高品质的前端代码，确保多平台兼容性和效能。
3. 参与需求分析、设计评审和技术讨论，提供专业的意见 and 建议。
4. 持续优化和改进现有的前端架构和代码，提升用户体验和开发效率。

**岗位要求：**
1. 本科及以上学历，计算机相关专业。
2. 至少 **1 年以上**前端开发经验，熟悉 HTML、CSS、JavaScript 等前端技术。
3. 熟练掌握 **React、Vue、Angular** 等主流前端架构之一。
4. 拥有良好的代码风格，能够编写清晰、可维护的代码。
5. 具备良好的沟通能力和团队合作精神。
6. 有成功上线的 Web 产品开发经验者优先。

### 4. C端 AI 产品经理
**岗位职责：**
1. 负责多模态大模型相关创新产品的落地与全流程管理。
2. 与开发团队密切合作，确保设计方案准确落地。
3. 跟进用户反馈，持续优化产品设计。

**岗位要求：**
1. 本科及以上学历，具备至少 **1 年以上** AI 产品相关经验，有从 0 到 1 的 C 端产品经验者优先。
2. 产品逻辑清晰，拥有优秀的原型设计与展示能力，具备良好的沟通、协调与组织能力。
3. 深入了解 **AIGC 和多模态大模型**技术及市场趋势，熟悉主流生成式 AI 产品并能分析其特点。
4. 具备同理心，对用户心理有深刻洞察，能从为用户创造价值的角度思考与解决问题。

### 5. 海外 AI 产品增长运营
**岗位职责：**
1. 负责运营主流媒体平台和数字营销，制定产品增长方案并实施。
2. 提升社区活跃和用户参与，吸引新用户加入，实现用户留存和持续增长。
3. 负责产品的全链路运营增长。

**岗位要求：**
1. 本科及以上学历，有互联网行业运营经验，其中至少有 **1 年**产品运营相关工作经验，有 AI 产品运营经验者优先。
2. 有出色的英语读写能力，有海外学习工作经验者优先。
3. 深入了解产品运营与增长的各个方面，有 **PLG (Product-Led Growth)** 经验者优先。
4. 具备优秀的沟通协作能力以及较强的问题解决能力。
5. 有海外 C 端产品成功运营经验，熟悉 AI 产品生态增长策略者优先。

### 6. SEO 英文编辑实习生
**岗位职责：**
1. 协助撰写、更新和维护公司网站及海外平台的英文内容，学习并应用 SEO 优化标准。
2. 在指导下进行关键词研究和搜索趋势分析，支持内容优化和流量提升。
3. 协助跟踪和分析内容表现，参与 SEO 相关的数据收集与报告工作。

**岗位要求：**
1. 本科或研究生在读，英文/翻译/小语种等外语专业优先。
2. 具备 **CET-6/TEM-4/雅思 6.0** 及以上英语水平或同等能力。
3. 英语基础扎实，具备较强的英文写作与表达能力。
4. 线下办公，每周出勤 5 天优先考虑。表现优秀者有转正机会。
